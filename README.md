A Unisulistas é um projeto de um sistema distribuído criado para resolver um problema de falta de memória em um sistemas semelhante, porém acoplado,esse problema.
Este sistema foi produzido utilizando o Strapi para o backend e o HTML, CSS e Javascript para o frontend. Como esse sistema não foi utilizado nenhuma hospedagem externa por ser apenas um projeto da faculdade, 
o Strapi foi hospedado apenas em um notebook, e utilizado o Ngrok para endereçar o Strapi ao frontend, já que o Strapi roda no localhost e sem o Ngrok o Vercel (responsável pela hospedagem do frontend) não teria como 
conectar à uma rede externa ao endereço do Vercel onde está o backend.
